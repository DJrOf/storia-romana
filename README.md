# Timeline Storia Romana

Una timeline interattiva e grafica che ricostruisce gli eventi principali della storia romana, dalla fondazione di Roma (753 a.C.) alla caduta dell'Impero Romano d'Occidente (476 d.C.).

## Caratteristiche

- **Timeline interattiva verticale** con visualizzazione cronologica degli eventi
- **Filtri per secolo** per navigare facilmente tra i diversi periodi storici
- **Ricerca** per trovare eventi specifici
- **Zoom** per ingrandire o ridurre la visualizzazione
- **Modal dettagliato** per visualizzare informazioni complete su ogni evento
- **Design moderno e responsive** ottimizzato per desktop e mobile
- **Oltre 200 eventi storici** accuratamente documentati

## Tecnologie Utilizzate

- React 18
- Vite
- CSS3 con animazioni moderne
- Design responsive

## Installazione

```bash
# Installa le dipendenze
npm install

# Avvia il server di sviluppo
npm run dev

# Build per produzione
npm run build
```

## Utilizzo

1. **Filtra per secolo**: Usa il menu a tendina per filtrare gli eventi per secolo
2. **Cerca eventi**: Usa la barra di ricerca per trovare eventi specifici
3. **Zoom**: Usa i pulsanti + e - per ingrandire o ridurre la visualizzazione
4. **Visualizza dettagli**: Clicca su un evento per vedere i dettagli completi in un modal
5. **Scroll**: Scorri verticalmente per navigare attraverso i secoli

## Eventi Inclusi

La timeline include eventi da:
- **VIII secolo a.C.** - Fondazione di Roma, età regia
- **VII-VI secolo a.C.** - Fine della monarchia, inizio della Repubblica
- **V-IV secolo a.C.** - Espansione repubblicana, guerre sannitiche
- **III-II secolo a.C.** - Guerre puniche, espansione mediterranea
- **I secolo a.C.** - Guerre civili, fine della Repubblica, inizio dell'Impero
- **I-II secolo d.C.** - Età d'oro dell'Impero, Cinque Buoni Imperatori
- **III secolo d.C.** - Crisi del III secolo
- **IV secolo d.C.** - Tetrarchia, Costantino, divisione dell'Impero
- **V secolo d.C.** - Caduta dell'Impero d'Occidente
